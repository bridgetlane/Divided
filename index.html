Divided

  • High Concept: Attempt to keep the continuously spawning spores divided.

  • Player Experience Goals: The game should cause the player to concentrate and be frustrating.

  • Genre: Casual games.

  • Game play: Use the 'W' and 'S' keys to move the center spore up and down. When a small spore looks like it is about to cross the divide, be sure to hit it with the large spore you control to prevent it from crossing. This will save you from losing and having each side contaminated, but it will also cause more spores to spawn.
    System dynamics: You can stop spores from crossing the divide, but that will cause more spores to spawn.
    Core mechanic: You can move the large spore up and down.

  • Prototype goal: This prototype explores a lot of randomness, something I had not done yet.

  • Features: My game is unique because you can only move in one direction at a time.

  • Setting/Premise: You want to stop the spores from mixing.

  • Story: There is no story.

  • Target Audience: Casual gamers.

  • Play time: 3 minutes.

  • Strategies: Try to anticipate spore movements.
